Interface ApiBindings

This is the TypeScript interface for the C++ BotlinkApi bindings.

The BotlinkApi implementation of this interface may throw an exception for any method in the interface when an error occurs.

The default timeout for methods with a timeoutSeconds optional parameter is 10 seconds. See BotlinkApi::defaultTimeout in C++ SDK's api.h.

The BotlinkApi implementation opportunistically refreshes the tokens when any method is called.
